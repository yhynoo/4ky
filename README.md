# 4ky

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

## Description

4ky is a web interface for browsing the administrative proto-cuneiform documents from CDLI and performing simple statistical analysis on them. It is part of ongoing studies and will be expanded in the future.

## Access

Visit the [4ky Web App](https://www.fourky.onrender.com)
